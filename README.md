# PHP Login App with Docker

This is a simple PHP login and registration app using MySQL and Docker.

## 🚀 Features

- User registration & login
- MySQL database integration
- Dockerized setup
- phpMyAdmin optional support

## 📁 Folder Structure

php-login-app/
├── docker-compose.yml
├── Dockerfile
├── src/
│ └── public/
│ ├── config/
│ │ └── db.php
│ ├── index.php
│ ├── register.php
│ └── login.php


## 🐳 Running with Docker

### 1. Build and Run
docker-compose up --build
